# 野菜の成長管理iosアプリ

野菜ごとに撮った写真と大きさを管理する。

![demo動画memo](./demo/memoDemo.gif)
![demo動画viewer](./demo/viewerDemo.gif)


## 使用方法

1. 右上の追加をするから、管理したい野菜の名前を入力
2. 追加した野菜をタップして、写真を撮影
3. 登録ボタンをタップして、野菜の大きさを入力
4. 成長状況で野菜の大きさと撮影した写真を自由に閲覧できる
    - 画像をスライドすることで、画像の切り替えができる
